== Description ==

[Product Page](https://www.storelocatorplus.com/) | [Video](https://www.youtube.com/channel/UCJIMv63upz-qIaB5EcursyQ)  | [Demo](https://demo.storelocatorplus.com) | [User Guide](https://docs.storelocatorplus.com) | [AddOns](https://wordpress.storelocatorplus.com) | [Managed Service](https://my.storelocatorplus.com)

A free add-on pack for the [Store Locator Plus](https://www.storelocatorplus.com/) location mapping plugin.
This add-on assists in cleaning up the Store Locator Plus settings including clearing out all pre-existing settings that may
interfere with upgrading or installing on a new server after restoring a WordPress backup.   Allows Store Locator Plus add-ons
to behave as if the plugin activation is happening on a new install.

= Options Clean Up =

Delete all [Store Locator Plus](https://www.storelocatorplus.com/) settings from the WordPress options table while retaining location data.

Supported Add On Packs:

* [Experience](https://wordpress.storelocatorplus.com/product/experience/)
* [Power] (https://wordpress.storelocatorplus.com/product/power/)
* [Premier](https://wordpress.storelocatorplus.com/product/premier-subscription/)


= Description Decoding =

Prior versions of Store Locator Plus may have encoded HTML while storing the data in the location descriptions.
A one-time-use tool will repair the location descriptions, converting the HTML encoded data to standard HTML
notation.   If you are seeing text such as &lt;br/&gt; instead of <br/> in your description data, you need this tool.

== Installation ==

= Requirements =

* Store Locator Plus: 4.9
* Wordpress: 4.4
* PHP: 5.4

= Install After SLP =

1. Install Store Locator Plus.
2. Download the latest .zip file for this plugin.
3. Go to plugins/add new.
4. Select upload.
5. Upload the zip file.
